关于幻影计划

幻影计划::MIRAGE 是一个文本隐写工具集，将敏感信息加密为看似正常的内容，以穿透关键词过滤系统。

本项目与宗教无关。灵感来源：与佛论禅

支持的模式：

  • 喵星密码 - 将文本转为喵星语拟声词组合
  • 火星文 - 使用异体字替换常见汉字
  • 乱码视界 - 在字符间穿插装饰符号
  • 瞎子算命 - AES加密 + 天干地支编码
  • 与佛论禅 - AES加密 + 佛经字符编码
  • 狗子汪汪 - 4进制汪语编码
  • 点阵盲文 - Unicode盲文点阵编码
  • 核心价值观 - Base8核心价值观编码
  • 纯Emoji - 静态emoji一一映射
