# 幻影计划::MIRAGE

一个文本隐写工具集，将敏感信息加密为看似正常的内容，以穿透关键词过滤系统（微信/微博等）。

## Statement 声明

This project has nothing to do with religion. 本项目与宗教无关。

## 功能

支持 9 种免 LLM 的静态伪装算法：

| 模式 | 说明 |
|------|------|
| 喵星密码 | 4进制喵语编码（喵/咪/呜/嗷），适合短文本穿透关键词过滤 |
| 火星文 | 异体字字典替换，需人工联系上下文 |
| 乱码视界 | 符号装饰穿插，可通过去符号还原 |
| 瞎子算命 | AES加密 + 天干地支编码，原文以"老衲掐指一算："开头 |
| 与佛论禅 | AES加密 + 佛经字符编码，原文以"佛曰："开头 |
| 狗子汪汪 | 4进制汪语编码，以"🐾"开头结尾 |
| 点阵盲文 | Unicode盲文点阵编码，肉眼难辨 |
| 核心价值观 | Base8核心价值观编码，伪装为社会主义核心价值观宣传语 |
| 纯Emoji | 静态emoji一一映射，以"🤐"开头结尾 |

## 使用

直接在浏览器打开 `docs/index.html` 即可使用。

## 原理

所有算法均基于以下流程：

1. **LZString 压缩** — 将原文压缩为 base64 字符串
2. **编码映射** — 将 base64 字符映射为特定字符集（如喵星语、盲文点阵等）
3. **解码还原** — 接收方粘贴密文，系统自动还原原文

AES 模式（瞎子算命/与佛论禅）额外使用 CryptoJS AES 加密，提供可配置密钥的保护。
